Shadow DOM New

This is component is using the shadow dom. While it is not supported in most browsers yet, we use it here to demonstrate how dynamic theming (doesn't) work with shadow dom.

  • Shadow DOM provides style encapsulation
    it doesn't inherit styles from the shadow host, nor does it inluence the shadow host with its own styles
  • Shadow DOM does inherit CSS Variables
  • Shadow DOM does inherit styling from the hosting body
  • New specs are on their way to allow for theming elements in the shadow dom (::part and ::theme)
  • It will require a significant change from UI libraries to come up with an approach to support shadow DOMs
